# BULLS AND COWS

## Game Rules

- A four-number code is chosen randomly
- Each turn, the player guesses a code
- Each digit of the code can either be a BULL or a COW

```yaml
BULL: that digit is correct, in the correct spot
COW: that digit is correct, but in the incorrect spot
```

- However, the player does not know which digits are BULLS or COWS
- They only know the amount of each one.
